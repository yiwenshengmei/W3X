W3X
---

## 项目介绍

* W3X是一个集抓取XXX页面、解析XXX页面、保存XXX页面上的图片、文字、种子并生成浏览种子、图片、文字的程序。
* W3X使用Java编写，将来有意移植到Android上。
* W3X因为使用Java编写，可配合定时器（比如Quartz）部署在WEB服务端定时运行。
